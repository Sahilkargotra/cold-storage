# 🚀 Quick Start Guide

## Step 1: Open the Application

1. Make sure the dev server is running
2. Open your browser
3. Go to: **http://localhost:5173/**

You should see:
```
┌─────────────────────────────────────────────────────┐
│  [CS] Cold Storage Management                      │
│       Intelligent Warehouse Operations              │
│                                     [Role Switcher ▼]│
└─────────────────────────────────────────────────────┘
```

## Step 2: Select Facility Dashboard

Click the dropdown and select: **Facility Dashboard**

This switches you to Ravi's view (Facility Manager at Chennai)

## Step 3: Explore Interactive Features

### 🎯 Activity 1: Resolve a Critical Alert

**What you'll see:**
```
┌─────────────────────────────────────────────────┐
│ 🔴 CRITICAL ALERTS              3 Open         │
│─────────────────────────────────────────────────│
│                                                 │
│ [🌡️] Zone B temperature at -12°C...     [OPEN] │
│      2025-01-15 09:30  [Zone B]                │
│      ₹45,000 at risk                            │
│                                                 │
│      (Click this alert to see details)  →       │
└─────────────────────────────────────────────────┘
```

**What to do:**
1. ✅ Click on the temperature alert
2. ✅ A modal opens showing:
   - Alert details
   - Financial impact (₹45,000)
   - Resolution notes text area
3. ✅ Type: "Technician dispatched. Compressor sensor replaced. Temperature back to -18°C"
4. ✅ Click: **"Resolve Alert"**
5. ✅ Alert is now grayed out with **RESOLVED** badge
6. ✅ Click "Show History" button to see your action logged!

**Result:** You saved ₹45,000! 🎉

---

### 🎯 Activity 2: Escalate an Alert

**What to do:**
1. ✅ Click on another alert (e.g., "Compressor #3 abnormal cycles")
2. ✅ Click: **"Escalate to Regional"** button
3. ✅ Alert status changes to **ESCALATED TO REGIONAL**
4. ✅ Blue badge shows escalation level
5. ✅ Escalation time logged

**What this means:**
- Alert is now forwarded to Priya (Regional Manager)
- She will see it in her dashboard
- Multi-level escalation in action! 📤

## 🎯 Key Things to Notice

### ✨ Interactive Elements
- **Clickable alerts** - Opens detail modal
- **Action buttons** - Approve, Reject, Resolve, Escalate
- **Generate buttons** - Create reports instantly
- **Toggle history** - Show/hide action history

### 💰 Financial Impact
- Every action shows ₹ value
- Savings highlighted in green
- Costs shown in red
- Running total in history

### 📊 Status Badges
- **OPEN** (Red) - Needs attention
- **IN_PROGRESS** (Yellow) - Being worked on
- **RESOLVED** (Green) - Completed
- **ESCALATED** (Blue) - Forwarded up

### 🎭 Role-Based Views
- Each role sees different data
- Different actions available
- Different level of detail
- Appropriate for decision-making
